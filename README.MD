# CRP 控制点标注工具（直方图自动阈值版）

该工具是一款基于 PySide6 开发的交互式图像控制点标注工具，支持批量处理图像，通过自动计算 ROI 区域直方图阈值（OTSU 算法）生成二值化掩膜，提取最大连通分量并计算重心作为控制点坐标。工具具有现代化图形界面，支持屏幕自适应、滚动按钮栏等功能。

## 一、项目环境

### 1. 依赖包

```
opencv-python==4.8.0.74
numpy==1.23.5
pyyaml==6.0.1
PySide6==6.7.0
```

### 2. 环境配置

推荐使用 Anaconda 创建虚拟环境（项目默认使用名为 CRP 的 conda 环境）：

```bash
# 创建 conda 环境
conda create -n CRP python=3.8

# 激活环境
conda activate CRP

# 安装依赖
pip install -r requirements.txt
```

## 三、使用方法

### 1. 运行程序

在项目根目录执行：

```bash
python main.py
```

### 2. 操作流程

程序启动后会显示主界面，按照以下步骤进行标注：

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 点击「选择图像」按钮 | 选择要标注的输入图像 |
| 2 | 点击「选择目录」按钮 | 选择输出目录（保存控制点坐标和标注图片） |
| 3 | 左键拖拽框选 ROI | 在图像上框选目标区域，自动生成二值化掩膜 |
| 4 | 右键点击微调掩膜 | 添加/移除掩膜区域（Shift+右键移除） |
| 5 | 输入控制点编号 | 在输入框中填写控制点编号 |
| 6 | 点击「确认控制点」按钮 | 计算重心并保存控制点坐标 |
| 7 | 点击「保存并退出」按钮 | 保存标注结果并退出 |

### 3. 交互操作说明

| 操作 | 功能说明 |
|------|----------|
| 左键拖拽 | 标注矩形框(ROI)，实时显示半透明蓝色矩形 |
| 右键点击 | 添加掩膜（补充漏选区域） |
| Shift+右键 | 移除掩膜（删除多余区域） |
| Ctrl+滚轮 | 缩放图像 |
| 中键拖拽 | 平移图像 |

### 4. 界面特点

- **屏幕自适应**：窗口自动适配屏幕分辨率（初始为屏幕的 90%）
- **最小尺寸限制**：窗口最小为 1024 × 600，确保控件正常显示
- **滚动按钮栏**：当控制面板内容超出屏幕高度时，自动显示垂直滚动条
- **实时状态显示**：底部状态栏显示当前操作状态和鼠标位置

## 四、输出结果

### 1. 控制点坐标文件（[图片名]_control_points.txt）

格式为：

```
控制点编号，像点 x 坐标，像点 y 坐标
1, 128.5678, 256.1234
2, 300.4567, 180.9876
```

### 2. 标注后可视化图片（[图片名]_labeled.jpg）

在原图上叠加最终掩膜和控制点编号，便于核对标注结果。

## 五、核心功能细节

- **自动阈值计算**：基于 OTSU 算法对框选的 ROI 区域计算二值化阈值，无需手动设置
- **最大连通分量提取**：自动过滤掩膜中的噪声区域，仅保留面积最大的连通区域
- **坐标映射**：图像缩放时自动完成窗口坐标与原图坐标的转换
- **掩膜微调**：支持手动添加/删除掩膜区域
- **亚像素级重心计算**：掩膜重心坐标采用均值计算，保留四位小数
- **撤销功能**：支持撤销上一次掩膜修改操作

## 七、注意事项

1. 建议使用分辨率较高的显示器以获得更好的标注体验
2. 标注前请确保已正确设置输出目录
3. 控制点编号建议按顺序递增，便于后续数据处理
4. 标注完成后建议核对输出图片确保标注准确性
