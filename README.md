## 原理简介

### 什么是 DLT？

直接线性变换（DLT）是近景摄影测量中的一种核心算法，它建立了**像点坐标 (x, y)** 与**物方空间坐标 (X, Y, Z)** 之间的线性关系，无需事先知道相机的内外方位元素。

### 数学模型

单像片的 DLT 方程可表示为：

```
x = (L₁X + L₂Y + L₃Z + L₄) / (L₉X + L₁₀Y + L₁₁Z + 1)
y = (L₅X + L₆Y + L₇Z + L₈) / (L₉X + L₁₀Y + L₁₁Z + 1)
```

其中 L₁ ~ L₁₁ 为 DLT 系数（共11个），包含：
- 内方位元素（主点 x₀, y₀，焦距 f）
- 外方位元素（摄站位置 Xₛ, Yₛ, Zₛ，姿态角 φ, ω, κ）
- 相机畸变参数（径向畸变 k₁, k₂, k₃，畸变 p₁, p₂）

### 完整处理流程

```
┌─────────────┐    ┌─────────────┐
│  控制场坐标  │    │  左片像点   │
│ (X,Y,Z)     │    │  量测数据   │
└──────┬──────┘    └──────┬──────┘
       │                  │
       └────────┬─────────┘
                ▼
        ┌───────────────┐
        │  DLT系数求解  │◄──── 使用6个以上控制点
        │  L₁ ~ L₁₁    │
        └───────┬───────┘
                ▼
        ┌───────────────┐
        │ 内/外方位元素  │
        │  畸变系数     │
        └───────┬───────┘
                ▼
        ┌───────────────┐
        │  前方交会      │◄──── 双片同名点匹配
        │  计算待定点   │
        └───────┬───────┘
                ▼
        ┌───────────────┐
        │  三维坐标     │
        │  (X, Y, Z)   │
        └───────────────┘
```

### 畸变校正模型

本系统采用附有畸变参数的 DLT 模型：

```
dx = x'·(k₁r² + k₂r⁴ + k₃r⁶) + p₁(r² + 2x'²) + 2p₂x'y'
dy = y'·(k₁r² + k₂r⁴ + k₃r⁶) + p₂(r² + 2y'²) + 2p₁x'y'

其中：
  x' = x - x₀, y' = y - y₀（去主点后的像点坐标）
  r² = x'² + y'²
  k₁, k₂, k₃：径向畸变系数
  p₁, p₂：偏心畸变系数
```

## 输入输出格式

### 输入文件格式

#### 控制场坐标文件（`.txt`）

```
点号 X Y Z
```

- **点号**：整数点号
- **X, Y, Z**：物方空间坐标（单位：mm）
- 点号 > 110 视为控制点（GCP），否则为待定点或检查点

#### 像点量测文件（`.txt`）

```
点号 类型 x y
可通过https://github.com/Hoo-U-Heon/Close-Range-Photogrammetry-Target-Marker进行像点量测
```

- **点号**：整数点号，需与控制场文件对应
- **类型**：0=待定点，1=控制点，2=检查点
- **x, y**：像素坐标（原点在图像左上角）

### 输出文件说明

| 文件名 | 内容 |
|--------|------|
| `dlt_左片.txt` / `dlt_右片.txt` | L系数、内/外方位元素、畸变系数、精度统计 |
| `check_point_verification.txt` | 检查点残差及外精度统计 |
| `unknown_points.txt` | 待定点三维坐标及到参考点距离 |

## 技术实现

### 算法核心模块

#### 1. `calculate.py` - DLT主计算

- `getInit()`：使用SVD从6个控制点初估L系数
- `getLi()`：迭代精化L系数（含畸变参数）
- `getElements()`：从L系数提取内/外方位元素
- `getUnknow()`：前方交会计算待定点坐标
- `getExtPre()`：检查点精度验证

#### 2. `DLT.py` - 空间后方交会

- `resection()`：基于共线条件的迭代平差
- `getR()`：计算旋转矩阵 R(φ, ω, κ)
- 同步解算14个未知数：外方位6个 + 内方位3个 + 畸变5个

#### 3. `data.py` - 数据管理

- `PhotoPt`：像点数据结构（含坐标转换）
- `LParams` / `RParams`：DLT参数与方位元素存储
- `readGCP()` / `readMeasure()`：文件读取

### 坐标转换

像素坐标 → 像平面坐标（mm）：

```python
# 图像尺寸：4272 × 2848 像素
# 传感器尺寸：22.2 × 14.8 mm
x_mm = (x_pixel - 4272/2) × (22.2/4272)
y_mm = (2848/2 - y_pixel) × (14.8/2848)
```

## 参数说明

### 内方位元素

| 参数 | 含义 | 单位 |
|------|------|------|
| x₀ | 主点x坐标 | mm |
| y₀ | 主点y坐标 | mm |
| f | 焦距 | mm |

### 外方位元素

| 参数 | 含义 | 单位 |
|------|------|------|
| Xₛ, Yₛ, Zₛ | 摄站位置 | mm |
| φ, ω, κ | 旋转角（φ=航偏角, ω=俯仰角, κ=横滚角） | 度 |

### 畸变系数

| 系数 | 含义 |
|------|------|
| k₁, k₂, k₃ | 径向畸变（2阶、4阶、6阶） |
| p₁, p₂ | 偏心畸变 |

### 精度统计

- **单位权中误差 σ**：反映观测值的整体精度
- **中误差**：各参数估计值的精度指标
- **检查点残差**：验证点计算值与真值的偏差

## 相机参数配置

相机参数集中定义在 `data.py` 文件顶部，方便统一修改。

### 配置参数

```python
# ==================== 相机参数配置 ====================
# 图像尺寸（像素）
IMAGE_WIDTH = 4272
IMAGE_HEIGHT = 2848

# CCD传感器尺寸（mm）
SENSOR_WIDTH = 22.2
SENSOR_HEIGHT = 14.8

# 像素大小（自动计算）
PIXEL_SIZE_X = SENSOR_WIDTH / IMAGE_WIDTH
PIXEL_SIZE_Y = SENSOR_HEIGHT / IMAGE_HEIGHT
# ====================================================
```

### 修改方法

如需更换相机，只需修改 `data.py` 中的上述常量：

```python
# 示例：更换为全画幅相机
IMAGE_WIDTH = 6000      # 新相机图像宽度
IMAGE_HEIGHT = 4000     # 新相机图像高度
SENSOR_WIDTH = 35.9     # 传感器宽度（mm）
SENSOR_HEIGHT = 24.0    # 传感器高度（mm）
```

所有相关计算（像素到毫米转换、精度统计等）会自动更新。
